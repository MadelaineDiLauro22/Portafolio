Pokedex

Mirá tus pokemones atrapados

Descripción

Como parte esencial de mi trayectoria en Programación Web 2 en la universidad, destaco un proyecto significativo denominado 'Pokedex'. Desarrollado con PHP, CSS, Bootstrap y JavaScript, esta aplicación encarna mi capacidad para integrar tecnologías esenciales en el desarrollo web.

La aplicación 'Pokedex' presenta un diseño atractivo y funcional, aprovechando las capacidades de Bootstrap y JavaScript. En lugar de recurrir a tecnologías adicionales, mi enfoque se centra en ofrecer una experiencia de usuario envolvente y efectiva sin agregar capas innecesarias de complejidad.

Este proyecto ilustra mi capacidad para crear aplicaciones web modernas, demostrando un equilibrio exitoso entre la innovación técnica, el diseño atractivo y la experiencia del usuario, utilizando únicamente PHP, CSS, Bootstrap y JavaScript.
