Descripción

En el marco de la materia Taller Web 1 en la Universidad Nacional de La Matanza, desarrollamos una aplicación web con Java, Spring Boot, Thymeleaf y MVC. Destacamos por un diseño atractivo gracias a Bootstrap, una experiencia interactiva con WebSocket y la integración de la API de Bing Maps. Nuestra aplicación refleja eficazmente los principios del desarrollo web moderno, abordando aspectos clave desde la estructura sólida con Spring Boot hasta la integración de servicios externos para una experiencia completa y avanzada.
